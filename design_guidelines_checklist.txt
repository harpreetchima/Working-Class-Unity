# Organization Design Guidelines & Assets Checklist

Please fill out or provide files for the following items to ensure we can accurately reflect your organization's brand on the new website.

## 1. Brand Identity
- [ ] **Logo Files:**
    - Preferred format: SVG (vector) for best quality.
    - Alternates: High-resolution PNG with transparent background.
    - Variations: Full logo, icon/mark only, horizontal vs. vertical versions, light vs. dark mode versions.
- [ ] **Color Palette:** (Required for DaisyUI theme configuration)
    - **Primary Brand Colors:** (Hex codes or OKLCH values, e.g., #0052CC)
    - **Secondary/Accent Colors:** (Hex codes or OKLCH values)
    - **Neutral Colors:** (Backgrounds, text colors, borders) - Do you have specific shades (50-950) or just base colors?
    - **Semantic Colors:** (Success, Warning, Error colors) - If not specified, we will use DaisyUI defaults.
- [ ] **Typography (Fonts):** (Required for TailwindCSS font family configuration)
    - **Primary Font:** (For headings/titles) - Include font files (WOFF2/TTF) or Google Fonts links.
    - **Secondary Font:** (For body text) - Include font files or links.
    - **Font Weights:** Specific weights needed? (e.g., Light 300, Regular 400, SemiBold 600, Bold 700)

## 2. Visual Style
- [ ] **Imagery Style:**
    - Do you use photography, illustrations, or 3D graphics?
    - Are photos candid/natural or staged/corporate?
    - Do images require any specific filters or treatments?
- [ ] **Iconography:**
    - Do you have a specific icon set? (e.g., outlined, filled, hand-drawn)
    - If not, do you prefer a specific style (e.g., FontAwesome, Heroicons)?
- [ ] **Shape & Form:** (Configures Tailwind `--radius-*` and `--shadow-*`)
    - **Border Radius:**
        - [ ] Sharp corners (0px)
        - [ ] Slightly rounded (4px - 8px)
        - [ ] Medium rounded (12px)
        - [ ] Fully rounded/Pill shapes
    - **Shadows:**
        - [ ] Flat design (no shadows)
        - [ ] Subtle depth
        - [ ] Pronounced shadows

## 3. UI Components & Layout
- [ ] **Buttons:** (Maps to DaisyUI button components)
    - **Primary Style:** Solid color, Outline, or Ghost?
    - **Shape:** Matches global border radius or specific (e.g., Pill-shaped)?
    - **Effects:** Gradients, animations on hover?
- [ ] **Spacing:** (Configures Tailwind `--spacing-*` scale)
    - Do you prefer a spacious, airy layout (comfortable) or a dense, information-heavy layout (compact)?

## 4. Content & Assets
- [ ] **Key Brand Assets:** Link to a folder (Dropbox/Drive) containing team photos, office photos, or product shots.
- [ ] **Favicon:** The small icon shown in the browser tab.

## 5. Voice & Tone
- [ ] **Tone:** Professional, playful, authoritative, friendly, technical, etc.
- [ ] **Taglines/Slogans:** Any specific phrases that must appear on the site?

## 6. Inspiration & Preferences
- [ ] **Reference Websites:** Links to 2-3 websites you like visually (and *why* you like them).
- [ ] **Dislikes:** Things you definitely *do not* want to see (e.g., "no carousels," "no orange buttons").
