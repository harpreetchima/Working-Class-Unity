# Pagination

Use these Tailwind CSS pagination components to add page navigation controls to your application's resource pages, featuring page numbers, next/previous buttons, and ellipses for handling large amounts of pages. These components are designed and built by the Tailwind CSS team, and include a variety of different styles and layouts.
