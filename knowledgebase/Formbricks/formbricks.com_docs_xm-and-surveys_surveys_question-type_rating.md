# Rating

Rating questions allow respondents to rate questions on a scale

Rating questions allow respondents to rate questions on a scale. Displays a title and a rating scale consisting of a number of images and labels for the lower and upper ends of the scale.

## [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/rating\#elements)  Elements

![Overview of Rating question type](https://mintcdn.com/formbricks/PoRbKmWNh09zaWpR/images/xm-and-surveys/core-features/question-type/rating.webp?fit=max&auto=format&n=PoRbKmWNh09zaWpR&q=85&s=f6a5f530c276610fdbd7a7ab77a14f51)

### [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/rating\#title)  Title

Add a clear title to inform the respondent what information you are asking for.

### [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/rating\#description)  Description

Provide an optional description with further instructions.

### [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/rating\#scale)  Scale

Select the icon to be used for the rating scale. The options include: stars, numbers or smileys. The default is stars.

### [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/rating\#range)  Range

Select the range of the rating scale. the options include: 3, 4, 5, 7 or 10. The default is 5.

### [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/rating\#labels)  Labels

Add labels for the lower and upper bounds of the rating scale. The default is “Not good” and “Very good”.
