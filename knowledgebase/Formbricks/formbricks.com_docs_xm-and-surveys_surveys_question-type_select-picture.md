# Picture Selection

Picture selection questions allow respondents to select one or more images from a list

**Self-Hosting Requirements**: This question type requires file upload storage to be configured for image
uploads. If you’re self-hosting Formbricks, make sure to [configure file\\
uploads](https://formbricks.com/docs/self-hosting/configuration/file-uploads) before using this feature.

Picture selection questions allow respondents to select one or more images from a list. Displays a title and a list of images for the respondent to choose from.

### [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/select-picture\#title)  Title

Add a clear title to inform the respondent what information you are asking for.

### [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/select-picture\#description)  Description

Provide an optional description with further instructions.

### [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/select-picture\#images)  Images

Images can be uploaded via click or drag and drop. At least two images are required. Each image must be 5 MB or less.
