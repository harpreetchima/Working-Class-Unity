# Ranking

Ranking questions let respondents select options in order from 1 to the total number of options. As they make their choices, the list is automatically rearranged in numerical order.

## [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/ranking\#elements)  Elements

![Overview of Ranking question type](https://mintcdn.com/formbricks/PoRbKmWNh09zaWpR/images/xm-and-surveys/core-features/question-type/ranking.webp?fit=max&auto=format&n=PoRbKmWNh09zaWpR&q=85&s=86a63fcf5217f1fd8170e16c56ac3096)

### [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/ranking\#title)  Title

Add a clear title to inform the respondent what information you are asking for.

### [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/ranking\#description)  Description

Provide an optional description with further instructions.

### [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/ranking\#options)  Options

You need to add at least two options so that users can rearrange them in numerical order based on their selection.

### [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/ranking\#select-ordering)  Select ordering

- Keep current order: This will keep the order of options the same for all respondents.
- Randomize all: This will randomize the options for each respondent.
