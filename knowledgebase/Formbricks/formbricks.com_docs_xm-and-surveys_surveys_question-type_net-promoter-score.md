# Net Promoter Score

Net Promoter Score questions allow respondents to rate a question on a scale from 0 to 10. Displays a title and a list of radio buttons for the respondent to choose from.

## [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/net-promoter-score\#elements)  Elements

![Overview of Net Promoter Score question type](https://mintcdn.com/formbricks/PoRbKmWNh09zaWpR/images/xm-and-surveys/core-features/question-type/net-promoter-score.webp?fit=max&auto=format&n=PoRbKmWNh09zaWpR&q=85&s=cad1287566e2c2e2c40cb412433f9723)

### [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/net-promoter-score\#title)  Title

Add a clear title to inform the respondent what information you are asking for.

### [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/net-promoter-score\#description)  Description

Provide an optional description with further instructions.

### [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/net-promoter-score\#labels)  Labels

Add labels for the lower and upper bounds of the score. The default is “Not at all likely” and “Extremely likely”.

### [​](https://formbricks.com/docs/xm-and-surveys/surveys/question-type/net-promoter-score\#add-color-coding)  Add color coding

Add color coding to the score. This will show a color bar above the score.
